# My Bank

## 마이 뱅크를 소개합니다.
마이 뱅크는 저의 소비패턴을 분석하기 위해 진행한 사이드 프로젝트입니다. 

## 프로젝트 환경
- java 17
- spring 3.2.0
- gradle 8.4
- maria DB 12.3.0
- postman API
- thymeleaf

## 프로젝트 개요
- 기능 구현 계획
  - 개인적으로 필요한 기능을 사용하기 위해 만든 프로젝트이기 때문에 최소한의 기능 구현으로 진행할 계획입니다.
- 확장성 고려
  - 단기 목표는 개인적으로 사용할 가계부지만 결과물의 편의성이 좋다고 판단되는 경우 프로젝트 규모를 확장할 수 있도록 설계하였습니다.
